pub async fn get_allowed_groups( &self, transaction: &mut PgConnection ) -> Result<Option<Vec<String>>, ModelError>

Return a list of allowed groups for a given network. Admin group should always be included. If no allowed_groups are specified for a network then all devices are allowed. In this case None is returned to signify that there’s no filtering. This helper method is meant for use in all business logic gating access to networks based on allowed groups.
